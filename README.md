CRUD Product Dashboard
📋 About

A fully responsive product management dashboard built with HTML, CSS, and JavaScript. The system allows users to efficiently manage product data through a complete CRUD workflow with real-time calculations and search functionality.

✨ Key Features

➕ Create, update, and delete products dynamically
🔍 Search products by title or category
💰 Automatic price calculation (taxes, ads, and discounts)
📊 Interactive table for product management
⚡ Responsive design for all devices
🧠 Clean and efficient vanilla JavaScript logic

🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
